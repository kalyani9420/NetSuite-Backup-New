Host key type - type of algorithm we use to generated SSH key pair
Host key - Public key of bank
Port - Port no of bank

2.Certifcate Keys - Select Private Key

Outbound Folder Name

WinSCP is use to generate SSH key
1.Select tool
2.Run puttygin
3.Select RSA in key
4.Enter No of bits #HOST key type and bits should be similar at both side NS & Bank while generating public keys
5.Click generate
6.save public key with .pub extension
7.We can enter Key passphrase to set password on private key files. No extension need to specify & .ppk will append bydefault
8.But NS don't accept private key with .ppk extension. Hence,Go to Load > Conversion > Open SSH key > This will generate private key with .pem extension
9.Name of Private key should not conataion space instead we can user '_'


Set up of private key in Keys of NS :

Setup > Company > Keys
1.New Key
2.Choose Private Key file (.pem) #Same key can share with multiple banks
